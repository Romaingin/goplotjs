## PlotLine

(x _[]float64_, y _[]float64_, name _string_)

Plot a function of `x` given `y`, with label `name`. Dimensions must agree.


## AddPlotLine
(x _[]float64_, y _[]float64_, name _string_)

Add a function of `x` given `y`, with label `name` to the previous line plot. Dimensions must agree.
